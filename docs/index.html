Key Features

  • C++20 modules for faster compilation and better encapsulation
  • OpenCV 4.x integration for computer vision operations
  • Thread-safe asynchronous processing pipeline
  • Real-time performance metrics (FPS, latency tracking)
  • Multiple object detection algorithms (Motion, HOG, DNN)
  • Kalman filter-based multi-object tracking
  • Stereo vision with depth estimation
  • Camera calibration and rectification

Component Interactions

  • Main thread orchestrates the processing pipeline
  • Image loader uses generators for lazy loading
  • Thread-safe queue handles inter-thread communication
  • Thread pool manages parallel processing
  • Detector and Tracker operate on separate worker threads
  • Performance metrics track latency in real-time

Pipeline Stages

  • Initialization: Setup detector, tracker, and thread pool
  • Image Loading: Load images from directory or generate synthetic frames
  • Queue Processing: Thread-safe queue for inter-thread communication
  • Object Detection: Motion-based, HOG, or DNN detection
  • Object Tracking: Kalman filter-based multi-object tracking
  • Visualization: Draw bounding boxes and track information
  • Metrics Collection: Track FPS and latency in real-time

Build Commands

  • Install vcpkg and OpenCV: ./vcpkg install opencv4[contrib]:x64-windows
  • Configure: cmake -B build -S . -DCMAKE_TOOLCHAIN_FILE=path/to/vcpkg.cmake
  • Build: cmake --build build --config Release
  • Copy DLLs: Copy OpenCV DLLs to build directory (Windows only)
  • Run: ./build/perception_demo.exe
